Library of Babel

The Library of Babel is an experimental publication with type only. All the content comes from the short story The Library of Babel, written by Jorge Luis Borges. After learning the fun fact that the author Jorge Luis Borges were influence a lot by a Jewish mysticism called Kabbalah, which believes that human can get the message from God by decoding the Bible, I decided to make my own code book.

If you follow the random page number and find one word on each page, you shall get this message: For every sensible line of straightforward statement, there are leagues of senseless cacophonies, verbal jumbles and incoherences.
